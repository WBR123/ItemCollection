What are the technologies

网页设计和网站开发用到的技术有哪些?

网站开发中使用的到网页bai设计的内容，所以我就以网站开发其中穿插网页设计为例进行说明

网站开发涉及前后端内容，而网页设计仅仅涉及前端展示的内容

所以网页设计中需要知道用的技术有HTML CSS JavaScript 等基础技术以及BootStrap 等框架

网站开发不仅包含以上部分，而且还包含有后台内容以及数据库的相关内容

后台的开发有多种方案和语言 例如Java PHP Python 等语言，每种语言又包含有各种框架

数据库同时又包含有SQL Server MySQL Oracle 等关系型和非关系型数据库几类
